Spartan Code - MVC with a twist.

Spartan code is a nice name for an approach to MVC-Pull in the land of web development frameworks. It is meant to ease development by keeping the programmer focused and the planning by keeping things separate. Altogether awesome results ahead!

The Spartan Code process is easy:

1. Make mockup
2. Code the program

Spartan for PHP

The first implementation for Spartan Code is in PHP. Its an app starter that gives you awesome templating, easy query management, URI routing and some helpers. There are no libraries included but you can use basically anything that you fancy!

Spartan Templating for Wordpress

This is a port of the PHP framework as a WP plugin. After it is installed it allows you to create WP themes that use the awesome spartan templating for both WP functionality and your own custom things. It also gives you the choice to encapsulate in models your WP theme specific work!

Spartan Templating for Code Igniter

The CI port is a helper file that allows skipping controllers and using the models as the sole source of data and logic. As always with spartan you'll need only one controller. The Views will parse Spartan comments and pull their data from the models directly.
